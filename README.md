       Weather App

A simple and responsive Weather Application that displays real-time weather data using the OpenWeather API.
FEATURES:
1. Search weather by city

2. Shows:

   Temperature

   Humidity

   Weather description

   Weather icon

3. Dynamic background changes (clear, cloudy, rainy, snow)

4.Loading spinner while fetching data

5. Press Enter to search

6. Clean and modern glassmorphism UI

TECHNOLOGIES USED:

1. HTML5

2. CSS3

3. JavaScript (Fetch API)

4. OpenWeather API

HOW TO USE:
1. Enter any city name in the search box.

2. Press Enter or click Search.

3. The app will show real-time weather information.

4. Background will change based on weather conditions.

How to Run Locally
1. Download or clone this repository

2. Add your OpenWeather API key in script.js

3. Open index.html in your browser

4. Enjoy the Weather App!
